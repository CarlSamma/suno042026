Pipeline a 5 Stadi

Ogni stadio utilizza un provider LLM specializzato per ottimizzare aspetti specifici della generazione del prompt

Pronto per iniziare
1

Deep Analysis

Google/Mistral

Compito: Analisi approfondita delle caratteristiche musicali

Input: The Beatles - Yesterday

Output: Ballata pop britannica del 1965, arrangiamenti orchestrali...

Tempo: 2-3s

2

Technical Refinement

Mistral

Compito: Raffinamento parametri tecnici musicali

Input: Dati analisi Beatles

Output: 90 BPM, tonalità C major, progressioni armoniche...

Tempo: 1-2s

3

Creative Enhancement

DeepSeek

Compito: Miglioramento aspetti creativi e atmosferici

Input: Specifiche tecniche + contesto musicale

Output: Strings orchestrali, voce espressiva, atmosfera nostalgica...

Tempo: 2-3s

4

Quality Control

Anthropic

Compito: Verifica conformità restrizioni Suno AI

Input: Prompt creativo migliorato

Output: Prompt validato: no nomi artisti, <400 caratteri...

Tempo: 1s

5

Final Generation

OpenAI

Compito: Generazione prompt finale ottimizzato

Input: Elementi prompt validati

Output: Melodic orchestral pop ballad, 90 BPM...

Tempo: 2s
